# J&J Real Estate Group — Official Website

Luxury real estate advisory website for **Chrystal Justice & Patsy Jenkins**, J&J Real Estate Group at Capstone Realty — Huntsville & North Alabama.

**Tagline:** Exceptional Properties. Extraordinary Results.

## Live Site

Hosted free on GitHub Pages. After enabling Pages (Settings → Pages → Deploy from branch → `main` → `/ (root)`), the site is live at:

```
https://<your-github-username>.github.io/<repo-name>/
```

## What's Inside

- `index.html` — the entire website (design, content, and interactivity in one file; no build step, no dependencies)
- `assets/` — optimized photos, headshots, and logos
- `HANDOFF_PROMPT.md` — full project brief so any developer or AI assistant can continue this work

## Editing Basics

Everything lives in `index.html`:

- **Phone numbers / emails:** search for `256-503-1924` or `choosecapstone.com`
- **Colors:** the `:root` block at the top of the `<style>` section (gold is `--gold: #c6a35d`)
- **Copy:** all text is plain HTML — find the sentence, change it
- **Photos:** replace files in `assets/` keeping the same filename, or update the `src` in the HTML
